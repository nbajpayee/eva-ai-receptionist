## ğŸ“ Important Notes

### Fonts
Currently using system fonts (sans-serif fallback). To enable Google Fonts:

1. Uncomment font imports in `app/layout.tsx`:
```typescript
import { Inter, DM_Sans } from "next/font/google";

const inter = Inter({ subsets: ["latin"], variable: "--font-inter" });
const dmSans = DM_Sans({ subsets: ["latin"], variable: "--font-display" });
```

2. Update `<html>` tag:
```typescript
<html className={`${inter.variable} ${dmSans.variable}`}>
  <body className={inter.className}>
```

This will work in any environment with internet access.

### Content Management
All site content is in `lib/constants.ts` for easy updates:
- Site metadata (name, description, contact info)
- Navigation items
- Features list
- Testimonials
- Pricing tiers
- FAQ items
- Integration partners

### Voice Demo Integration
To enable the actual voice interface:

1. Create `components/VoiceClient.tsx` based on `frontend/app.js`
2. Connect to WebSocket: `ws://localhost:8000/ws/voice/{session_id}`
3. Replace placeholder in `app/voice-demo/page.tsx`

See README.md section "5. Enable Voice Demo" for detailed instructions.
